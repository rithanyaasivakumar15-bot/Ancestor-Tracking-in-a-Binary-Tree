Ancestor Tracking in a Binary Tree

Overview

This project implements an Ancestor Tracking System in a Binary Tree using C programming. It demonstrates how hierarchical relationships can be represented and managed using a non-linear data structure. The system allows users to create a binary tree, insert nodes, search nodes, perform inorder traversal, and find ancestors of a specific node using recursion.

Features

* Create a root node dynamically
* Insert child nodes under specified parent nodes
* Search for nodes in the binary tree
* Perform inorder traversal
* Find ancestors of a given node
* Menu-driven user interaction
* Input validation for handling invalid operations

Concepts Used

* Binary Tree
* Structures in C
* Pointers
* Dynamic Memory Allocation
* Recursion
* Tree Traversal

How It Works

The program creates a binary tree where each node contains data and pointers to its left and right children. Users can build the tree dynamically by adding nodes. The ancestor tracking feature uses recursion to trace the path from the target node back to the root and displays all ancestor nodes.

Menu Options

* 1. Create root node
* 2. Insert node under parent
* 3. Print inorder traversal
* 4. Print ancestors of a node
* 5. Exit

Technologies Used
